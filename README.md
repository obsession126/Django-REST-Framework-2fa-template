Django 2FA API

Бекенд-проєкт на Django + Django REST Framework з реалізацією реєстрації, входу та двофакторної автентифікації (2FA) через email-коди за допомогою brevo

Особливості

Реєстрація користувача – створення нового акаунту з email та паролем.

Вхід з 2FA – після введення пароля користувач отримує код підтвердження.

Кастомна модель користувача – логін через email.

Сесійна безпека – 2FA коди зберігаються на сервері.

Використані технології

Python 3.13

Django 5.2

Django REST Framework

SQLite (можна замінити на іншу БД)

dotenv для змінних оточення

Ендпоінти API
/api/auth/register/	POST	Реєстрація користувача
/api/auth/login/	POST	Вхід користувача + генерація 2FA коду
/api/auth/verify-2fa/	POST	Підтвердження коду 2FA
